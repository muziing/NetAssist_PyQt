## 简介

学习 PyQt5 过程中用来练习实践的项目。能实现基本的 TCP、UDP 连接与信息发送，以及一个简易 WebServer。

可以搭配博文食用：[网络调试助手 程序设计（PyQt5实战）——来自muzing的博客](https://muzing.top/posts/5ab16c09/)
